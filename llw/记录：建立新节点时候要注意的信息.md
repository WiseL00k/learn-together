- cmakelist
- package.xml 要有如下内容：
```xml
<export>
  <controller_interface plugin="${prefix}/simple_sync_controller_plugins.xml"/>
  <controller_interface plugin="${prefix}/simple_gimbal_controller_plugins.xml"/>
</export>
```
- 对应的 `plugins.xml` 要有如下内容：
```xml
<library path="lib/lib<name of the folder>>">
  <class name="<name_of_the_file/NameOfTheClass>"
         type="<name_of_the_file::NameOfTheClass>"
         base_class_type="<base_type_namespace::BaseType>">
    <description>
      Some description
    </description>
  </class>
</library>
```
注意替换所有 `<>` 中的内容
- hpp 中的 
```cpp
#ifndef <NAME_OF_THE_FILE>
#define <NAME_OF_THE_FILE>
```
