### 搭建Gazebo仿真五连杆机构

创建URDF文件，名为`five_link.urdf.xacro`

其中参考数据如下:

| 连杆 | 长度（单位：m） |
| ---- | --------------- |
| l1   | 0.14            |
| l2   | 0.25            |
| l3   | 0.25            |
| l4   | 0.14            |
| l5   | 0.12            |

关于gazebo中搭建闭链结构，可以采用gazebo标签解决。参考如下：

```xml
<gazebo>
    <joint name="connect_joint" type="revolute">
        <parent>l2_link</parent>
        <child>l3_link</child>
        <axis>
            <xyz>0 1 0</xyz>
        </axis>
        <pose>0 0 0.24 0 0 0</pose>
    </joint>
</gazebo>
```
